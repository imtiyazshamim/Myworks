Get Ready for the Holiday Season!

With the Holiday Season just around the corner that will push demands through the roof for retail & eCommerce, we bring you the release of GreyMatter's new versions plus Warehouse Operations guide & cheat sheets to be ready for the peak season.

New Release - GreyMatter v5.4.9 & v5.0.13

The new GreyMatter versions add a major update to our fulfillment operating system that powers the Ranger™ Series. It will help our customers improve system usability, make manageability simpler, help scale up to 750 bots per warehouse, and provide better stability through algorithmic improvements.

Scaling New Heights - Go Beyond 200k units/day

The Ranger™ enabled warehouse can now cater to larger operations. It can process more than 200k units in a single day and run ~750 Rangers to support the supply chain network from a centralized aWarehouse™.

Better Pick Capacity with Latest Station Mgmt.

Focusing on quality-of-life improvements for the Operations team, we have completely revamped the Station Management in GreyMatter Command Center™(GCC). The enhanced user experience makes it more intuitive for users to make changes. It also brings the much-awaited Hierarchical Bin tagging capability, which will allow the system to adjust to the changing order patterns and evenly distribute pick capacity.

Enhanced Manageability & Inventory Visibility

The GCC is going to have a single management interface for Ranger Chargers™. Plus, you will have more visibility of the audit process and the ability to generate filter-based reports. We have also revamped the Inventory Listing page, bringing a live inventory tracker view. It enables users to track inventory aging and take actions like audit, recall and quantity/tag change.

Manage Omnichannel Orders with Real-time PPS Recommendations

The brand-new PPS Setting Manager™ is built keeping in mind omnichannel operations. It recommends changes in PPS settings like mode, status, profile & queue lengths based on shifts and current order pool.

Algorithmic Improvements for Omnichannel

The new GreyMatter™ Pick Algorithm will improve the Picks from each MSU face to up to 10%. The algorithmic improvements cater to a true omnichannel scenario where the system processes orders based on order characteristics. The Ranger traffic management is also modified to reduce the risk of congestion when order volumes are high.

Operations' Playbooks & Tool-kits

Introducing a mind map that will help you understand all the parameters affecting pick UPH on a site and thus, enables you to find out inefficiencies, eliminate them and run operations smoothly.

This guide will prepare our operations team to manage daily site operations. It has a checklist for monitoring operations, handling exceptions, special scenarios, and taking proactive and reactive measures like how to recover from system failures or change inventory status. The playbook also shares insights on steps to be followed for various operations-related tasks, recoveries etc.

The out-of-the-box reports and dashboards, mapped to various personas, give a snapshot view of the last 24 hours to 15 days of operations, system performance as well as KPIs. It also shows historical trends of KPIs and can be subscribed on a daily or weekly basis.
