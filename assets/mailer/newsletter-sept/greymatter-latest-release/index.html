GreyMatter Latest Release -
v5.4.9 & v5.0.13
Get Ready for the Holiday Season!

With the Holiday Season just around the corner that will push demands through the roof for retail & eCommerce, we bring you the release of GreyMatter's latest version. These versions add a major update to our fulfillment operating system that powers the Ranger™ Series. It will help our customers improve system usability, make manageability simpler, help scale up to 750 bots per warehouse, and provide better stability through algorithmic improvements.

Scaling New Heights

The Ranger™ enabled warehouse now can cater to larger scale operations. It can:

  • Process more than 200k units in a single day.
  • Run ~750 Rangers to support the supply chain network from a centralized aWarehouse™.

Simplified Station Management

Focusing on quality-of-life improvements for our Operations team, we have completely revamped the user experience for the Station Management in GreyMatter Command Center™(GCC).

We have introduced the much-awaited Hierarchical Bin tagging capability. It will allow the system to adjust and evenly distribute pick capacity as per the changing order distribution patterns.

Enhanced Manageability

The GCC is going to provide a single management interface for Ranger Chargers™. We have also revamped the Inventory Listing page, bringing a live inventory tracker view. It enables users to track inventory aging and take actions like audit, recall and tag change.

New PPS Manager

The brand-new PPS Setting Manager™ is built keeping in mind omni-channel operations. It recommends changes in PPS settings like mode, status, profile & queue lengths based on shifts and current order pool.

Leveraging Inventory-in-Motion™ and more

The new GreyMatter™ Pick Algorithm will improve the Picks from each MSU face to up to 10%. The algorithmic improvements cater to a true omnichannel scenario where the system processes orders based on order characteristics. It auto-tunes itself to process eCommerce orders to focus more on SLA adherence and retail orders to achieve higher productivity.

Boosting the Audit

The new capability gives the complete visibility of the audit process, of the orders dependent/blocked due to it, and details like audit creator, executor, approver, date & time. Furthermore, it allows users to create filters reports and also regenerate all the rejected audits. Apart from this, the page's performance has greatly improved with faster filters, search and very low latency in load times.

Enhanced UI response and Ranger™ Traffic Mgmt.

A better UI response times, speedy and accurate Manager Dashboard™ data synchronization, and new Ranger™ Traffic Management module is introduced to give the end-user a navigation seamless experience for quick decision-making. It will reduce the risk of congestion when order volumes are high.
